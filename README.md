# AdventureWorks-Sales-Analytics
End-to-End Business Intelligence project using SQL Server and Power BI.

## Project Overview

This project demonstrates an end-to-end Business Intelligence solution using Microsoft SQL Server and Power BI.

Starting from the AdventureWorks2022 OLTP database, I designed and implemented a dimensional data warehouse, built an ETL process, created analytical SQL queries, and developed an interactive executive dashboard for business decision-making.

---

## Business Problem

The AdventureWorks management team requires a centralized reporting solution to monitor:

- Sales performance
- Product profitability
- Customer behavior
- Territory performance
- Executive KPIs

The objective is to transform operational data into meaningful business insights through a star-schema data warehouse and interactive Power BI dashboards.

## Business Insights

Some of the insights generated include:

- Southwest is the highest-performing sales territory.
- Bikes generate the highest revenue.
- Store customers contribute significantly more revenue than individual customers.
- Product profitability varies considerably across categories.
- A small percentage of customers generate the majority of total revenue.
